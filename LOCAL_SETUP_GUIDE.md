# 윈도우 10 전용 서버 설정 가이드

이 가이드는 현재 준비하신 윈도우 10 컴퓨터(깡통 컴퓨터)를 보고서 관리 시스템의 **전용 서버**로 만드는 방법을 안내합니다.

---

## 1단계: 필수 프로그램 설치 (Node.js)

프로그램을 실행하기 위한 엔진인 **Node.js**가 필요합니다.

1. [Node.js 공식 홈페이지](https://nodejs.org/ko/)에 접속합니다.
2. **"LTS" (Stable)** 버전을 다운로드하여 설치합니다.
3. 설치가 끝나면 '명령 프롬프트(CMD)'를 열고 다음을 입력하여 확인합니다:
   ```cmd
   node -v
   ```
   (버전 숫자가 나오면 성공입니다!)

---

## 2단계: 환경 변수 설정 (`.env.local`)

프로그램이 데이터베이스와 보고서 폴더를 찾을 수 있게 주소를 알려줘야 합니다.

1. 프로젝트 폴더의 `.env.local` 파일을 엽니다. (없으면 새로 만듭니다)
2. 다음 내용을 복사해서 붙여넣습니다 (값은 본인의 값으로 수정):
   ```env
   # 데이터베이스 연결 정보 (Supabase)
   NEXT_PUBLIC_SUPABASE_URL=당신의_URL
   NEXT_PUBLIC_SUPABASE_ANON_KEY=당신의_ANON_KEY
   SUPABASE_URL=당신의_URL
   SUPABASE_SERVICE_ROLE_KEY=당신의_SERVICE_ROLE_KEY

   # 보고서 폴더 경로 (중요!)
   REPORT_STORAGE_ROOT=Z:\data\측정팀\측정보고서
   ```

---

## 3단계: 프로그램 실행

1. '명령 프롬프트' 또는 '파워쉘'에서 프로젝트 폴더로 이동합니다.
2. 처음 실행한다면 다음 명령어로 필요한 것들을 다운로드합니다:
   ```cmd
   npm install
   ```
3. 서버를 시작합니다:
   ```cmd
   npm run dev
   ```
   (화면에 `ready - started server on 0.0.0.0:3000` 같은 메시지가 나오면 성공입니다!)

---

## 4단계: 접속하기

- **서버 PC에서**: 인터넷 주소창에 `http://localhost:3000`을 입력합니다.
- **다른 컴퓨터에서**: 서버 PC의 IP 주소를 알아낸 뒤 `http://[IP주소]:3000`으로 접속합니다.

---

## 5단계: 24시간 끊김 없이 유지하는 팁

1. **절전 모드 해제**: [설정] -> [시스템] -> [전원 및 절전]에서 '최대 절전 모드'와 '절전 모드'를 **안 함**으로 설정하세요.
2. **화면만 끄기**: 전력 절약을 위해 모니터는 꺼두셔도 프로그램은 계속 돌아갑니다.
