1. 개발 참고사항

A. Z:\data\측정팀\자동화 툴\MES 프로그램 DB에 다운되어 저장(기존 파일 덮어씀)된 2개의 파일(사업장정보.xls, 측정사업장.xls)을 신규 데이터베이스 파일로 만들고, 만들어진 데이터베이스 파일에서 참고하여 측정일지 데이터베이스를 새로 생성하여, HTML의 자료로 활용한다.

- 참고로 해당 경로에 있는 해당 파일의 분석을 돕기 위해 동일한 자료를 측정일지_Html 폴더에 첨부한다.

B. 대략 10여명의 사용자가 데이터에 접근하여 사용할 예정이며, html의 생성된 탭을 이용하여 자료를 조회, 수정, 저장하는 자동화를 실현한다.

C. 개발, 유지 비용 등 별도의 비용이 발생되지 않도록 구현하며, 비용 발생 우려 시 검토 내용을 상세히 설명한다.

3. UI 화면 : 모든 화면은 엑셀로 다운 받을 수 있는 기능이 구현되어야 한다.

A. 대시보드
1) 측정건수와 미완료 건수
2) 측정완료일 기준 25일 경과 사업장명과 경과일수
3) 지정한계_관할지청 사업장 수
4) 매출현황(측정비와 기타매출로 분리하여 표시)
5) 년도별 및 월별 매출 추이
6) 미수관리 : 년도별, 주기별 검색 --> 미수 총 금액, 미수 건수
7) 추가로 필요한 사항

B. 예비조사 탭
1) 사업장.XLS 자료를 활용한다.
2) 항목 : 측정일, 종료일, 측정요일, 사업장명, 측정자, 공시료코드, 주소, 예비조사자, 실측정자, 보고서
3) 항목 구현 설명
- 측정일 : mm/dd로 표시 (예 : 20260101 또는 0101 입력 시 01/01로 변환하여 출력
- 종료일 : 측정일 값을 기본갑으로 하고, 수정할 수 있도록 할 것
- 측정요일 : 측정일과 종료일으로 요일 명시. 만약 측정일과 종료일이 범위 값인 경우에는 공휴일을 제외한 요일을 모두 기록
- 사업장명 : 사업장정보의 자료의 데이터를 가져오는데, 사업장정보의 데이터에 없는 신규인 경우도 등록할 수 있도록 구현
- 측정자 : 이태환, 한기문, 강종구, 이주형, 배윤민, 고유빈
- 공시료 코드 : 측정자가 입력되면 자동으로 부여하도록 구현하는데, 측정자가 2인인 경우 첫번째 측정자를 기준으로 코드 부여
(측정자별 코드 : 이태환 --> A, 한기문 --> B, 강종구 --> C, 이주형 --> D, 배윤민 --> E, 고유빈 --> F
- 주소 : 사업장정보의 주소1 값을 자동으로 가져오게 구현
- 예비조사자 : 측정자 중 복수 선택할 수 있도록 목록에서 체크하여 등록할 수 있도록 구현
- 실측정자 : 측정자 중 목록에서 체크하여 등록할 수 있도록 구현
- 보고서 : 측정자 중 목록에서 체크하여 등록할 수 있도록 구현
주의) 입사, 퇴사자 발생 시 측정자 쉽게 수정할 수 있도록 구현

C. 측정일지 탭
1) 절차 : 검색 기능으로 정보 검색 --> 선택 --> 수정 --> 저장
2) 검색 기능 : 측정년도, 측정주기, 사업장명, 지정한계_관할지청, 주소로 검색이 가능해야 하며, 직전(가장 최신 자료)의 자료를 불러와야 한다. 다만, 사업장명에 *번외*가 포함된 기록된 자료는 검색된 자료에서 제외한다.
3) 저장 시 자동으로 측정년도, 측정주기를 변경하여 저장하는데, 측정년도와 측정주기가 변경되지 않았을 경우에는 경고창을 띄워서 사용자의 입력오류를 최소화하도록 한다.
4) 측정사업장 자료의 완료여부에 "미완료"일 경우로 경우로 변경되면 측정일지 데이터도 변경되도록 한다. 다만, 완료 이후에는 자료가 변경되면 안된다.
5) 소재지 관할청
측정사업장의 주소를 기준으로 "노동지청별 관할지역.csv"와 매칭되는 노동지청 자동 입력
6) 지정한계_관할지청
소재지 관할청을 참고하여 대전지방고용노동청, 중부지방고용노동청 평택지청, 중부지방고용노동청 경기지청은 해당 지청으로 하고, 기타지청은 대전지방고용노동청 천안지청 그룹
7) 공문연번
- 대전지방고용노동청 천안지청 : 천-001 부터 순차적으로 부여
- 대전지방고용노동청 : 대-001 부터 순차적으로 부여
- 중부지방고용노동청 평택지청 : 평-001부터 순차적으로 부여
- 중부지방고용노동청 경기지청 : 경-001부터 순차적으로 부여
8) 연번 : 상반기, 하반기로 나눠서 순차적으로 부여한다. 다만, 관한지청별로 중복되면 안됨(예: 001, 002, 003……)
9) 5인 이상 연번
지정한계_관할지청로 상∙하반기를 구분하여 부여하는데, “측정사업장.xls”의 총인원 필드를 참조하는데, 순차적으로 번호를 부여한다. 다만, 5인 미만의 경우에는 직전에 사용한 번호를 그대로 부여(중복될 수 있음)한다.(중요)
예) 천안지청 기준으로 A사업장 3명, B사업장 8명, C사업장 5명, D사업장 4명인 경우 A사업장 000, B사업장 001, C사업장 002, D사업장 002 이러한 순서로 부여되어야 함.
