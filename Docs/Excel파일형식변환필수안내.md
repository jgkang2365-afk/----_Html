# Excel 파일 형식 변환 필수 안내

## 문제 상황

업로드된 Excel 파일이 `.xls` 형식(Excel 97-2003)인 경우, `xlsx` 라이브러리가 파일을 제대로 파싱하지 못할 수 있습니다.

오류 메시지 예:
- `Bad ExternSheet: 2582 != 8`
- `Excel 파일 파싱 실패`

## 해결 방법

### Excel에서 .xlsx로 변환 (필수)

1. Excel에서 업로드할 파일 열기
   - `측정사업장.xls` 또는 `사업장정보.xls`
   
2. 파일 형식 변환
   - "파일" → "다른 이름으로 저장"
   - 파일 형식: **"Excel 통합 문서 (*.xlsx)"** 선택
   - 파일명: 기존 이름 유지 (예: `측정사업장.xlsx`)
   
3. 변환된 파일 업로드
   - 대시보드에서 변환된 `.xlsx` 파일 업로드

## 참고사항

- `.xls` 파일은 오래된 Excel 형식으로, 최신 라이브러리와 호환성 문제가 있을 수 있습니다
- `.xlsx` 형식으로 변환하면 모든 기능이 정상적으로 작동합니다
- 변환 후에도 데이터는 동일하게 유지됩니다
