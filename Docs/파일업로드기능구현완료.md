## 사용 방법

### 1. Supabase Storage 설정

1. Supabase 대시보드 → Storage
2. 새 버킷 생성: `excel-files`
3. RLS 정책 설정 (자세한 내용은 `Docs/SETUP_SupabaseStorage설정가이드.md` 참고)

### 2. 파일 업로드

1. 대시보드 페이지 접속
2. "Excel 파일 업로드" 섹션에서:
   - 파일 타입 선택 (사업장정보 / 측정사업장)
   - Excel 파일 선택
   - "파일 업로드" 버튼 클릭
3. 업로드 완료 후 수동 동기화 실행

### 3. 동기화

1. "Excel 파일 동기화 상태" 섹션에서
2. "수동 동기화" 버튼 클릭
3. **Storage에 파일이 있으면 Storage의 최신 파일을 사용**
4. Storage에 파일이 없으면 로컬 파일 시스템의 파일을 사용 (개발 환경)

## 동작 방식

- **Storage 우선**: 파일 경로가 지정되지 않으면 Storage에서 최신 파일을 먼저 확인
- **로컬 파일 Fallback**: Storage에 파일이 없으면 로컬 파일 시스템 사용 (개발 환경)
- **명시적 경로**: 파일 경로가 지정된 경우 해당 경로의 파일 사용
