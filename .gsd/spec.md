# Specification - Mobile Modal Resizing

본 문서는 모바일 환경에서 모달의 크기를 사용자가 자유롭게 조정할 수 있도록 하는 기능에 대한 명세서입니다.

## 1. 목적
- 현재 데스크탑 환경에서만 작동하는 모달 리사이징 기능을 모바일(터치 환경)에서도 지원하도록 확장.
- 모바일 사용자가 상세 데이터를 확인할 때 화면 공간을 효율적으로 활용할 수 있도록 함.

## 2. 주요 기능
- **터치 기반 리사이징**: 우측 하단의 리사이즈 핸들을 터치하여 드래그할 때 모달의 너비와 높이가 실시간으로 변경됨.
- **최소 크기 보장**: 리사이징 시 모달이 너무 작아지지 않도록 최소 너비(300px) 및 최소 높이(200px) 제한 유지.
- **반응형 핸들**: 터치 조작이 용이하도록 리사이즈 핸들의 터치 영역 최적화.

## 3. 기술적 요구사항
- `Modal.tsx` 컴포넌트 내에 `TouchEvent` 핸들러 추가 (`touchstart`, `touchmove`, `touchend`).
- `touchmove` 이벤트 발생 시 `e.touches[0]`를 사용하여 좌표 계산.
- 기존 `MouseEvent` 로직과 터치 로직을 통합하여 코드 중복 최소화.
- 모바일 브라우저의 기본 스크롤 동작 방해 금지 (리사이징 중에만 스크롤 방지).

## 4. UI/UX 디자인
- 리사이즈 핸들 아이콘은 기존 디자인 유지하되, 터치 감도 향상을 위해 투명한 `hit-area` 확장.
- 리사이징 중에는 모달의 `transition` 효과를 일시적으로 제거하여 부드러운 조작감 제공.

## 5. 검증 계획
- iOS/Android 브라우저 환경에서 터치 드래그를 통한 모달 크기 변경 확인.
- 리사이징 중 내용물이 올바르게 스크롤되는지 확인.
- 리사이징 후 닫기 버튼 및 내부 입력 요소들의 상호작용 정상 작동 확인.
