# [Spec] 모달 자동 닫힘 버그 수정

## 1. 개요
측정대상 사업장 목록 관리 모달에서 입력 필드(비고 등)에 입력 시 모달이 자동으로 닫히는 현상을 수정합니다.

## 2. 문제 정의
- **원인**: `Modal` 컴포넌트의 브라우저 히스토리 관리 로직이 `onClose` 함수의 변경을 감지하여 `useEffect`를 재실행하고, 이 과정에서 `window.history.back()`이 호출되어 모달이 닫히는 사이드 이펙트가 발생함.
- **영향**: 사용자가 모달 내에서 데이터를 입력하거나 수정할 수 없음 (입력할 때마다 모달이 닫힘).

## 3. 해결 설계
- `components/ui/Modal.tsx`의 `useEffect` 로직 개선.
- `onClose` 함수를 `useRef`로 래핑하여 의존성 배열에서 제거.
- `isOpen` 상태 변화에만 반응하도록 히스토리 로직 격리.

## 4. 변경 사항 (예정)
### `components/ui/Modal.tsx`
- `onCloseRef` 추가 및 `useEffect`로 업데이트.
- 히스토리 관련 `useEffect` 의존성 배열에서 `onClose` 제거.
- Escape 키 관련 `useEffect` 의존성 배열에서 `onClose` 제거.
