# [Spec] 대시보드 미발행 업체 목록 - 측정일지 상세 연동 자동화

## 1. 개요
대시보드의 '계산서 미발행 업체 목록'에서 업체명을 클릭했을 때, 해당 업체의 측정일지 화면이 새 창으로 열리고 자동으로 검색 및 상세 수정 모달까지 활성화되어 사용자의 작업 동선을 최소화한다.

## 2. 주요 기능 및 변경 사항

### A. 대시보드 고도화 (`components/features/Dashboard.tsx`)
- **업체명 클릭 이벤트**: 
  - 단순 텍스트에서 클릭 가능한 링크 형태(Cursor Pointer, Hover Underline)로 변경.
  - 클릭 시 `window.open`을 사용하여 `/journal` 페이지를 새 탭으로 호출.
  - URL 파라미터 전달: `code` (업체코드), `year` (측정년도), `period` (측정주기), `autoOpen=true` (자동 팝업 여부).

### B. 측정일지 검색 엔진 개선 (`components/features/JournalSearch.tsx`)
- **URL 파라미터 처리**:
  - 컴포넌트 마운트 시 URL의 `code`, `businessName`, `year`, `period` 등을 감지하여 검색 조건(`.searchParams`) 초기화.
- **자동 검색 및 팝업 (One-Step UI)**:
  - 파라미터가 존재할 경우 즉시 검색(`handleSearch`) 실행.
  - 검색 결과가 **정확히 1건**이고 `autoOpen=true`인 경우, `handleSelectJournal`을 자동으로 호출하여 수정 모달을 즉각 렌더링.

## 3. 성공 DNA 및 규칙 준수
- **정확도 우선**: 업체명 중복 가능성을 고려하여 반드시 **업체코드(`code`)**를 우선 순위로 검색 필터링에 참여시킨다.
- **데이터 일관성**: 대시보드에서 전달하는 `year`, `period` 정보가 측정일지의 데이터와 일치하도록 동기화 로직을 유지한다.

## 4. 기대 효과
- 대시보드 확인 -> 업체명 클릭 -> 즉시 일지 수정 가능한 상태로 진입 (클릭 횟수 3회 -> 1회로 단축).
- 담당자별 미발행 건 처리에 대한 생산성 대폭 향상.
