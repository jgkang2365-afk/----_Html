# Specification: 모바일 브라우저 뒤로가기 시 모달 닫기 개선

## 1. 개요
모바일 환경에서 모달이 열린 상태로 브라우저 '뒤로가기'를 누를 경우, 현재 페이지가 유지되지 않고 대시보드(이전 페이지)로 바로 이동하는 문제를 해결합니다. History API를 활용하여 뒤로가기 시 모달만 닫히도록 개선합니다.

## 2. 변경 사항
- **Modal.tsx (공통 컴포넌트)**
    - 모달이 열릴 때(`isOpen: true`) 가상의 히스토리 상태(`modalOpen: true`)를 `pushState`로 추가합니다.
    - 브라우저의 `popstate` 이벤트를 감시하여, 뒤로가기 발생 시 모달을 닫도록(`onClose` 호출) 처리합니다.
    - 모달이 닫힐 때(`isOpen: false`), 수동으로 닫힌 경우(X 버튼, 배경 클릭 등) 추가했던 히스토리 상태를 `history.back()`으로 정리합니다.

- [x] `popstate` 이벤트 리스너 등록 및 해제 처리

- [x] X 버튼 등으로 모달을 닫은 후 뒤로가기 클릭 시 정상적으로 이전 페이지(대시보드 등)로 이동하는지 확인
- [x] 여러 개의 모달이 중첩될 경우에 대한 동작 검증 (필요시)
